# ConsentGuard

AI privacy auditor landing page for Daytona HackSprint Berlin 2026.

## Open in VS Code

Open this folder:

```text
ConsentGuard
```

## Install

```powershell
npm install
```

## Run

```powershell
npm run dev -- --hostname 127.0.0.1 --port 3001
```

Then visit:

```text
http://127.0.0.1:3001
```

## Build

```powershell
npm run build
```
